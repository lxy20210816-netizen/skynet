# 金融指数数据

获取股票市场指数数据。

## 📚 脚本说明

### nasdaq_pe.py
获取纳斯达克指数PE（市盈率）数据。

### stock_indices.py
获取各种股票指数数据（上证、深证、纳斯达克、标普500等）。

## 🚀 使用方法

### 获取纳斯达克PE数据
```bash
python3 nasdaq_pe.py
```

### 获取股票指数数据
```bash
python3 stock_indices.py
```

## 📊 数据来源

- Yahoo Finance
- 各大交易所官方API

## 📂 输出位置

数据保存到: `~/Desktop/workspace/brain/skynet/`
